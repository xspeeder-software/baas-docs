# 日志查询

## 操作日志

系统实时记录每个操作员的操作日志，提供查询界面

![](http://static.toughcloud.net/toughsms/tc_20180517162520_11.png)


## 系统日志

系统实时记录每个模块的运行日志，记录了发生的关键信息，包括关键错误信息，比如查询某个用户的相关日志，可以直接输入用户账号作为关键字查询，系统日志会自动清理，也可以手工清理。

- 支持查询各个系统模块的运行日志
- 支持查询不同日志级别
- 关键词快速匹配，支持用户账号模糊匹配，快速定位用户故障
